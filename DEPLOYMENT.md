# Руководство по запуску TeleClone на личном ноутбуке

Это руководство поможет вам превратить исходные файлы в работающее приложение на вашем компьютере.

## 1. Требования
Убедитесь, что на вашем ноутбуке установлены:
- **Node.js** (версия 18 или выше) — это "двигатель", на котором работает приложение.
- **PostgreSQL** — это база данных, где будут храниться ваши сообщения.

## 2. Установка
1. Скачайте все файлы проекта в одну папку на ноутбуке.
2. Откройте терминал (командную строку) и перейдите в эту папку.
3. Установите все необходимые компоненты командой:
   ```bash
   npm install
   ```

## 3. Настройка базы данных
1. Создайте пустую базу данных в PostgreSQL (назовите её, например, `teleclone`).
2. Создайте в папке проекта файл с названием `.env` и добавьте в него следующие строки:
   ```env
   DATABASE_URL=postgresql://ЛОГИН:ПАРОЛЬ@localhost:5432/teleclone
   SESSION_SECRET=придумайте_любой_пароль_для_сессий
   ```
   *(Замените ЛОГИН и ПАРОЛЬ на те, что вы указали при установке PostgreSQL)*

## 4. Подготовка приложения
Выполните команду, чтобы подготовить таблицы в базе данных:
```bash
npm run db:push
```

## 5. Запуск приложения

### Постоянный запуск (для использования):
1. Сначала подготовьте файлы (сборка):
   ```bash
   npm run build
   ```
2. Запустите приложение:
   ```bash
   npm start
   ```
Приложение откроется в браузере по адресу: `http://localhost:5000`

## 6. Как общаться с друзьями (Приватность)
Приложение создано для 3-5 человек и работает без облачных серверов:
1. **VPN (Безопасный способ)**: Установите **Tailscale** или **ZeroTier** на свой ноутбук и ноутбуки друзей. Это создаст "невидимый кабель" между вами. Друзья смогут зайти в приложение, введя ваш внутренний IP-адрес.
2. **Peer ID**: Каждый пользователь получает уникальный код (Peer ID) в настройках. Чтобы начать чат, просто добавьте Peer ID друга в разделе «Контакты».

---
**Важно:** Все ваши переписки хранятся **только на вашем ноутбуке**. Никто другой, включая разработчиков или Google, не имеет к ним доступа.
