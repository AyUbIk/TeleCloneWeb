## Packages
peerjs | P2P connectivity for direct messaging
framer-motion | Smooth animations for messages and UI transitions
date-fns | Time formatting and relative dates
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes
react-textarea-autosize | Auto-expanding input field
uuid | Unique ID generation for messages and peers

## Notes
Tailwind Config: Extended colors for Telegram theme (sidebar, bubbles, accents).
LocalStorage: Primary data store for messages and chats (teleclone_* keys).
API: /api/gemini/chat is the only backend endpoint used; others are client-side simulated or P2P.
Fonts: Inter (sans-serif) as primary font.
Icons: Lucide React for all UI icons.
