# Пошаговая инструкция для новичка: Как запустить свой Telegram

Привет! Это простое руководство поможет тебе запустить приложение, даже если ты никогда не занимался программированием.

## Шаг 1: Подготовка (Нужно сделать один раз)

Тебе нужно установить две бесплатные программы на свой ноутбук:
1. **Node.js**: Скачай версию "LTS" с сайта [nodejs.org](https://nodejs.org/). Просто нажимай «Далее» при установке.
2. **PostgreSQL**: Это база данных. Скачай её с [postgresql.org](https://www.postgresql.org/download/). При установке **обязательно запомни пароль**, который ты укажешь!

## Шаг 2: Настройка файлов

1. Создай на рабочем столе папку, например `MyChat`.
2. Положи в неё все файлы, которые ты получил.
3. В этой папке создай обычный текстовый файл и назови его `.env` (обязательно с точкой в начале).
4. Открой его через Блокнот и вставь туда это:
   ```env
   DATABASE_URL=postgresql://postgres:ТВОЙ_ПАРОЛЬ@localhost:5432/teleclone
   SESSION_SECRET=любое_длинное_слово_на_английском
   ```
   *Вместо `ТВОЙ_ПАРОЛЬ` напиши тот пароль, который ты придумал при установке PostgreSQL.*

## Шаг 3: Первый запуск (Магия)

1. Открой папку с чатом.
2. В строке адреса (где написано название папки) сверху напечатай `cmd` и нажми Enter. Откроется черное окно (терминал).
3. Напечатай там по очереди эти команды:
   - `npm install` (подожди, пока всё скачается)
   - `npm run db:push` (создаст базу данных)
   - `npm run build` (подготовит приложение к работе)
   - `npm start` (запустит сам чат)

## Шаг 4: Как пользоваться

1. После команды `npm start` открой браузер и перейди по адресу: `http://localhost:5000`
2. Твой чат открыт! 
3. Зайди в **Настройки (Settings)**. Там ты увидишь свой **Peer ID**.
4. Скопируй этот ID и отправь другу. Друг должен сделать то же самое у себя.
5. Чтобы переписываться, нажми **Контакты -> Add** и вставь ID друга.

## Как пригласить друзей без интернета? (Приватность)

Чтобы друзья могли зайти к тебе, установите все вместе программу **Tailscale**. Она создаст вашу личную закрытую сеть. После этого друзья смогут зайти в твой чат, введя твой IP-адрес из Tailscale в браузере.

**Всё! Твои сообщения теперь только на твоем ноутбуке.**
